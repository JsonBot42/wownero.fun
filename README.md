## 🚀 How to Run

1.  **Clone the repo**:
    ```bash
    git clone https://github.com/your-username/wownero-landing.git
    cd wownero-landing
    ```

2.  **Open it**:
    Just double-click `index.html` or run a simple python server:
    ```bash
    python3 -m http.server
    ```

3.  **Visit**: `http://localhost:8000`
